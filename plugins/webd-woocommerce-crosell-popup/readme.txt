Boost Cross Sales with Upsells Modal Products Popup on Ajax Added to Cart with Woocommerce Upsell Plugin.
 
== Description ==

= Boost your Eshop cross sales with an upselling product popup window on Ajax Added to Cart! =
<p>An  Upsells & Cross Sells Popup plugin that once a user adds to cart a product, a popup window will appear with the defined upselling products.</p> 
<p>The plugin will work on Single Product's Page, it AJAXifies the add to cart button and displays in the popup window:</p>
* Links : View your Cart, Checkout
* Upselling Products : these must be already defined through product editing process
*  Ability to show or hide on mobile


**GET POPUP ON ARCHIVE AND SHOW RELATED PRODUCTS IF UPSELL NOT DEFINED WITH PRO VERSION**

= PRO Features =

* Ability to Show Related Products on Popup if Upsells have not been set - so plugin will work out of the Box
* Show Popup on Shop, Categories and Archive Pages. Ability to show or hide
* Choose what to Show in the Archive page. This is very useful in terms of if your eshop has thousands of products, setting Upsells for each product one by one is not an easy job. Thus, you can to show either Upsell or Crosell but Also RELATED that belong to same category and will be shown automatically.
*  Ability to show or hide on mobile
* Style popup: width, margin-top, border-radius

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce -> WooCommerce Crosell Popup to define if you need widget to appear on Mobile
4. Plugin is plug n play once activated it will work on single product page.
